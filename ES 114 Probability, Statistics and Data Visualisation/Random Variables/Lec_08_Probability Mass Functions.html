Probability Mass Functions (PMF)

Introduction to Random Variables​

A random variable (RV), denoted as XXX, is a function that maps outcomes from a sample space Ω\OmegaΩ to real numbers. Essentially, it translates "words" (events) into "numbers" for mathematical analysis.

For an outcome ξ∈Ω\xi \in \Omegaξ∈Ω, the random variable XXX assigns a numerical value X(ξ)X(\xi)X(ξ).

Key Idea: While a sample space can have many events, and a random variable can map these to many numbers (states), the goal is to systematically describe the probabilities associated with these numerical outcomes.

Probability Mass Functions (PMF)​

The probability mass function (PMF) of a discrete random variable XXX is a function that quantifies the probability of the random variable taking on a specific numerical value. We denote the PMF as pX(a)p_X(a)pX​(a).

Definition: For a discrete random variable XXX, its PMF is defined as: pX(a)=P[X=a]p_X(a) = P[X = a]pX​(a)=P[X=a] where P[X=a]P[X = a]P[X=a] is the probability of the event {ξ∈Ω∣X(ξ)=a}\{\xi \in \Omega \mid X(\xi) = a\}{ξ∈Ω∣X(ξ)=a}.

Distinction between XXX and aaa:

  • XXX: This is the random variable itself, a function from the sample space to numbers. Technically, it should be X(ξ)X(\xi)X(ξ).
  • aaa: This is a specific state or value that the random variable XXX can take. So, X=aX=aX=a means the random variable XXX takes on the state aaa.

A random variable is considered "random" precisely because it can adopt various states, each with an associated probability.

Example: Coin Flip​

Consider flipping a fair coin twice. Let XXX be the random variable representing the number of heads. The sample space is Ω={TT,TH,HT,HH}\Omega = \{TT, TH, HT, HH\}Ω={TT,TH,HT,HH}.

The PMF for XXX is:

  • pX(0)=P[X=0]=P["TT"]=14p_X(0) = P[X = 0] = P[\text{"TT"}] = \frac{1}{4}pX​(0)=P[X=0]=P["TT"]=41​ (0 heads)
  • pX(1)=P[X=1]=P["TH", "HT"]=24=12p_X(1) = P[X = 1] = P[\text{"TH", "HT"}] = \frac{2}{4} = \frac{1}{2}pX​(1)=P[X=1]=P["TH", "HT"]=42​=21​ (1 head)
  • pX(2)=P[X=2]=P["HH"]=14p_X(2) = P[X = 2] = P[\text{"HH"}] = \frac{1}{4}pX​(2)=P[X=2]=P["HH"]=41​ (2 heads)

The PMF values graphically represent the probability distribution across the possible states of XXX.

Histograms​

A histogram is a graphical representation of the distribution of numerical data. It groups data into bins and shows the frequency (or relative frequency/probability) of values falling into each bin.

For example, a histogram of letter frequencies in English shows the observed relative frequency (empirical probability) of each letter.

PMF vs. Histogram​

The relationship between a PMF and a histogram is profound:

  • A histogram is an empirical representation of data. It shows the observed frequencies or probabilities from a finite set of trials.
  • A PMF is the ideal or theoretical histogram. It represents the true underlying probability distribution of a random variable, often derived from a probabilistic model.

As the number of observations (NNN) in a histogram increases, the histogram tends to converge towards the true PMF of the underlying random variable. For instance, repeatedly rolling a fair die and plotting the histogram of outcomes will approach a uniform PMF of 1/61/61/6 for each face as the number of rolls approaches infinity.

Why Study PMF?​

  1. Ideal vs. Empirical: We seek an ideal mathematical description of randomness, not just empirical observations.
  2. Data Modeling: PMFs provide a parametric model for data, allowing us to understand the underlying process that generates the data, unlike non-parametric histograms.
  3. Data Generation and Testing: Once a PMF is known, we can synthesize new data that adheres to the established probability distribution. This is crucial for simulations, testing algorithms, and statistical inference.

Properties of a PMF​

A valid Probability Mass Function pX(x)p_X(x)pX​(x) must satisfy two fundamental properties:

Theorem: For a discrete random variable XXX with possible values x∈Xx \in \mathcal{X}x∈X (the support of XXX), its PMF pX(x)p_X(x)pX​(x) must satisfy:

  1. Non-negativity: pX(x)≥0p_X(x) \geq 0pX​(x)≥0 for all x∈Xx \in \mathcal{X}x∈X.
  2. Normalization: ∑x∈XpX(x)=1\sum_{x \in \mathcal{X}} p_X(x) = 1∑x∈X​pX​(x)=1.

Proof of Normalization: Let X\mathcal{X}X be the set of all possible values that XXX can take. The sum of probabilities for all possible values of XXX is: ∑x∈XpX(x)=∑x∈XP[X=x]\sum_{x \in \mathcal{X}} p_X(x) = \sum_{x \in \mathcal{X}} P[X=x]∑x∈X​pX​(x)=∑x∈X​P[X=x] By definition, P[X=x]P[X=x]P[X=x] is the probability of the event {ξ∈Ω∣X(ξ)=x}\{\xi \in \Omega \mid X(\xi)=x\}{ξ∈Ω∣X(ξ)=x}. Since these events are mutually exclusive (a random variable cannot take two different values simultaneously) and exhaustive (it must take some value), their union covers the entire sample space Ω\OmegaΩ. ∑x∈XP[X=x]=P[⋃x∈X{ξ∈Ω∣X(ξ)=x}]\sum_{x \in \mathcal{X}} P[X=x] = P\left[ \bigcup_{x \in \mathcal{X}} \{\xi \in \Omega \mid X(\xi)=x\} \right]∑x∈X​P[X=x]=P[⋃x∈X​{ξ∈Ω∣X(ξ)=x}] The union of all these disjoint events constitutes the entire sample space Ω\OmegaΩ. P[⋃x∈X{ξ∈Ω∣X(ξ)=x}]=P[Ω]P\left[ \bigcup_{x \in \mathcal{X}} \{\xi \in \Omega \mid X(\xi)=x\} \right] = P[\Omega]P[⋃x∈X​{ξ∈Ω∣X(ξ)=x}]=P[Ω] By the axioms of probability, the probability of the sample space is 1. P[Ω]=1P[\Omega] = 1P[Ω]=1 Thus, ∑x∈XpX(x)=1\sum_{x \in \mathcal{X}} p_X(x) = 1∑x∈X​pX​(x)=1.

Quick Summary​

A Probability Mass Function (PMF), pX(a)=P[X=a]p_X(a) = P[X=a]pX​(a)=P[X=a], defines the probability of a discrete random variable XXX taking on a specific numerical value aaa. Unlike an ordinary variable with a single state, a random variable can assume multiple states, each with a given probability. PMFs are the theoretical, "ideal" counterpart to empirical histograms, which they converge to with an increasing number of observations. Essential properties for any valid PMF are non-negativity (pX(x)≥0p_X(x) \geq 0pX​(x)≥0) and normalization (∑pX(x)=1\sum p_X(x) = 1∑pX​(x)=1), ensuring that probabilities are valid and sum to certainty over all possible outcomes.
