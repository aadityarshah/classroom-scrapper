Expectation of Random Variables

The concept of expectation is fundamental in probability theory, representing the "average" value of a random variable. It's not just a simple average but a weighted average, where each possible value of the random variable is weighted by its probability.

1. Defining Expectation​

The expectation (or expected value or mean) of a discrete random variable XXX, denoted as E[X]E[X]E[X], is a measure of the central tendency of the distribution.

Definition​

For a discrete random variable XXX with a sample space Ω(X)\Omega(X)Ω(X) and probability mass function (PMF) pX(x)p_X(x)pX​(x), the expectation is defined as:

E[X]=∑x∈Ω(X)xpX(x)E[X] = \sum_{x \in \Omega(X)} x p_X(x)E[X]=∑x∈Ω(X)​xpX​(x)

This definition implies summing over all possible values xxx that XXX can take, multiplying each value by its corresponding probability.

Interpretation: Weighted Average​

The expectation E[X]E[X]E[X] is a weighted average. Each state (value xxx) is weighted by its "percentage" or probability pX(x)p_X(x)pX​(x). This means values that are more probable contribute more significantly to the expectation.

For instance, if a dataset is partitioned into bins, and each bin has a value xkx_kxk​ and a percentage of samples pX(xk)p_X(x_k)pX​(xk​), the "average" is effectively the expectation.

2. Expectation vs. Sample Average (Histogram)​

It is crucial to distinguish between the theoretical expectation and the empirical sample average.

  • Expectation (E[X]E[X]E[X]):

    • A statistical property of the random variable itself.
    • A deterministic number derived from the PMF.
    • Often unknown; it's the target of estimation.
    • Calculated "top-down" using the known probability distribution.

  • Sample Mean (Xˉ\bar{X}Xˉ):

    • A numerical value calculated from a finite set of observed data.
    • Itself a random variable because it depends on the specific samples drawn.
    • Has uncertainty, which decreases as more samples are used.
    • Used to estimate the true mean (E[X]E[X]E[X]).
    • Calculated "bottom-up" from data, without requiring knowledge of the true underlying distribution XXX.

The Law of Large Numbers states that as the number of samples NNN increases, the sample mean Xˉ\bar{X}Xˉ will converge to the true expectation E[X]E[X]E[X]. Similarly, the empirical histogram of observed data will increasingly resemble the underlying probability mass function (PMF) as N→∞N \to \inftyN→∞.

4. Existence of Expectation: Absolutely Summable​

Does the expectation always exist? No. For E[X]E[X]E[X] to exist, the sum ∑xpX(x)\sum x p_X(x)∑xpX​(x) must converge. More formally, we require the random variable to be absolutely summable.

Definition: Absolutely Summable​

A discrete random variable XXX is absolutely summable if the expectation of its absolute value is finite: E[∣X∣]=def∑x∈X(Ω)∣x∣pX(x)<∞E[|X|] \overset{\text{def}}{=} \sum_{x \in X(\Omega)} |x| p_X(x) < \inftyE[∣X∣]=def∑x∈X(Ω)​∣x∣pX​(x)<∞ Only those absolutely summable random variables have finite expectations. If E[∣X∣]E[|X|]E[∣X∣] diverges, then E[X]E[X]E[X] does not exist or is undefined.

Consider a random variable XXX with the following PMF: pX(k)=6π2k2,k=1,2,…p_X(k) = \frac{6}{\pi^2 k^2}, \quad k = 1, 2, \dotspX​(k)=π2k26​,k=1,2,… Let's check if E[X]E[X]E[X] exists by evaluating E[∣X∣]E[|X|]E[∣X∣]: E[∣X∣]=∑k=1∞∣k∣pX(k)=∑k=1∞k6π2k2=6π2∑k=1∞1kE[|X|] = \sum_{k=1}^{\infty} |k| p_X(k) = \sum_{k=1}^{\infty} k \frac{6}{\pi^2 k^2} = \frac{6}{\pi^2} \sum_{k=1}^{\infty} \frac{1}{k}E[∣X∣]=∑k=1∞​∣k∣pX​(k)=∑k=1∞​kπ2k26​=π26​∑k=1∞​k1​ The sum ∑k=1∞1k\sum_{k=1}^{\infty} \frac{1}{k}∑k=1∞​k1​ is the harmonic series, which is known to diverge to infinity. Since E[∣X∣]=∞E[|X|] = \inftyE[∣X∣]=∞, the expectation E[X]E[X]E[X] for this random variable does not exist.

Quick Summary​

  • Expectation E[X]E[X]E[X] is the weighted average of a discrete random variable's possible values, where each value is weighted by its probability pX(x)p_X(x)pX​(x).
  • Formula: E[X]=∑x∈Ω(X)xpX(x)E[X] = \sum_{x \in \Omega(X)} x p_X(x)E[X]=∑x∈Ω(X)​xpX​(x).
