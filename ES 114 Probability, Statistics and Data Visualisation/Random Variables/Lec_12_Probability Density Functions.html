Probability Density Functions

Intuition: Probability as a Measure of Size​

For continuous sample spaces, we define the probability of an event AAA by measuring the "size" of AAA relative to the sample space Ω\OmegaΩ.

In the simplest case of equiprobable outcomes (Uniform distribution) over an interval:

P[{x∈A}]="size" of A"size" of Ω=∫Adx∫Ωdx=∫A1∣Ω∣dx\mathbb{P}[\{x \in A\}] = \frac{\text{"size" of } A}{\text{"size" of } \Omega} = \frac{\int_A dx}{\int_\Omega dx} = \int_A \frac{1}{|\Omega|} dxP[{x∈A}]="size" of Ω"size" of A​=∫Ω​dx∫A​dx​=∫A​∣Ω∣1​dx

Where ∣Ω∣|\Omega|∣Ω∣ is the length (or area/volume) of the sample space. To relax the equiprobable assumption, we replace the constant weight 1/∣Ω∣1/|\Omega|1/∣Ω∣ with a varying weight function fX(x)f_X(x)fX​(x).

Formal Definition of the PDF​

Let XXX be a continuous random variable. The Probability Density Function (PDF) fX:Ω→Rf_X: \Omega \to \mathbb{R}fX​:Ω→R is a mapping that, when integrated over an interval [a,b][a, b][a,b], yields the probability that XXX falls within that interval.

Axiomatic Properties​

  1. Non-negativity: fX(x)≥0f_X(x) \geq 0fX​(x)≥0 for all x∈Ωx \in \Omegax∈Ω.
  2. Unity (Normalization): ∫−∞∞fX(x)dx=1\int_{-\infty}^{\infty} f_X(x) dx = 1∫−∞∞​fX​(x)dx=1.
  3. Measure of a Set: P[X∈A]=∫AfX(x)dx\mathbb{P}[X \in A] = \int_A f_X(x) dxP[X∈A]=∫A​fX​(x)dx.

Key Interpretations​

  • Density vs. Probability: fX(x)f_X(x)fX​(x) is not the probability P[X=x]\mathbb{P}[X=x]P[X=x]. It represents probability per unit length.
  • Approximation: For small δ\deltaδ, P[x≤X≤x+δ]≈fX(x)⋅δ\mathbb{P}[x \leq X \leq x + \delta] \approx f_X(x) \cdot \deltaP[x≤X≤x+δ]≈fX​(x)⋅δ.
  • Magnitudes: Unlike PMFs, fX(x)f_X(x)fX​(x) can be greater than 1, and can even approach infinity, provided the integral remains equal to 1.

Point Probabilities and Intervals​

In continuous spaces, the probability of a single point is zero:

P[X=x]=∫xxfX(t)dt=0\mathbb{P}[X = x] = \int_x^x f_X(t) dt = 0P[X=x]=∫xx​fX​(t)dt=0

Implication: For any continuous random variable, the inclusion or exclusion of endpoints does not change the probability of the interval:

P[a≤X≤b]=P[a<X<b]=P[a≤X<b]=P[a<X≤b]\mathbb{P}[a \leq X \leq b] = \mathbb{P}[a < X < b] = \mathbb{P}[a \leq X < b] = \mathbb{P}[a < X \leq b]P[a≤X≤b]=P[a<X<b]=P[a≤X<b]=P[a<X≤b]

Example: PDF with a Singularity​

Consider fX(x)=12xf_X(x) = \frac{1}{2\sqrt{x}}fX​(x)=2x​1​ for 0<x≤10 < x \leq 10<x≤1, and 000 otherwise. As x→0x \to 0x→0, fX(x)→∞f_X(x) \to \inftyfX​(x)→∞. However, the total probability is still valid:

∫0112xdx=[x]01=1\int_0^1 \frac{1}{2\sqrt{x}} dx = \left[ \sqrt{x} \right]_0^1 = 1∫01​2x​1​dx=[x​]01​=1

Connecting PDF and PMF​

Discrete random variables can be represented in the continuous framework using the Dirac delta function δ(x)\delta(x)δ(x). A PMF can be viewed as a PDF consisting of a "train" of deltas.

General Delta Representation​

If XXX is a discrete random variable with PMF pX(xk)p_X(x_k)pX​(xk​), its PDF is:

fX(x)=∑xk∈ΩpX(xk)δ(x−xk)f_X(x) = \sum_{x_k \in \Omega} p_X(x_k) \delta(x - x_k)fX​(x)=xk​∈Ω∑​pX​(xk​)δ(x−xk​)

Discrete Examples in Continuous Form​

  1. Bernoulli RV: fX(x)=pδ(x−1)+(1−p)δ(x−0)f_X(x) = p\delta(x-1) + (1-p)\delta(x-0)fX​(x)=pδ(x−1)+(1−p)δ(x−0).
  2. Binomial RV: fX(x)=∑k=0n(nk)pk(1−p)n−kδ(x−k)f_X(x) = \sum_{k=0}^n \binom{n}{k} p^k (1-p)^{n-k} \delta(x-k)fX​(x)=∑k=0n​(kn​)pk(1−p)n−kδ(x−k).

Calculating Probability with Deltas​

When integrating a PDF containing deltas over an interval [a,b][a, b][a,b], the result is the sum of the weights of all deltas located within that interval:

P[a≤X≤b]=∫ab(∑pX(xk)δ(x−xk))dx=∑xk∈[a,b]pX(xk)\mathbb{P}[a \leq X \leq b] = \int_a^b \left( \sum p_X(x_k) \delta(x - x_k) \right) dx = \sum_{x_k \in [a, b]} p_X(x_k)P[a≤X≤b]=∫ab​(∑pX​(xk​)δ(x−xk​))dx=xk​∈[a,b]∑​pX​(xk​)

Quick Summary​

  • The PDF fX(x)f_X(x)fX​(x) defines the distribution of a continuous random variable; its integral over a set gives the probability.
  • Total area under the PDF curve must always be exactly 111.
  • Probability at any point xxx is 000 for continuous distributions.
  • fX(x)f_X(x)fX​(x) is a density, not a probability; it can exceed 111.
  • Discrete PMFs can be unified with the PDF framework using Dirac delta functions, allowing discrete and continuous variables to be treated with the same calculus-based tools.
