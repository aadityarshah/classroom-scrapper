Introduction to Random Variables

In probability theory, experiments often yield outcomes that are descriptive rather than inherently numerical (e.g., "Head," "Tail," "rainy," "sunny"). While these can be analyzed directly, mathematical operations like averaging or calculating variance become complex. Random variables serve as a crucial bridge, translating these descriptive outcomes into numerical values, thereby simplifying mathematical analysis.

1. Why We Need Random Variables​

Consider computing probabilities for events described verbally:

  • "Probability of getting a head": P["Head"]P[\text{"Head"}]P["Head"]
  • "Probability of getting 3 heads in 3 coin flips": P["Head"∩"Head"∩"Head"]P[\text{"Head"} \cap \text{"Head"} \cap \text{"Head"}]P["Head"∩"Head"∩"Head"]

This notation is often unwieldy. By introducing random variables, we assign numbers to outcomes. For instance, in a coin flip, we can define "Head" as 1 and "Tail" as 0. If XXX is the sum of heads:

  • P["Head"]P[\text{"Head"}]P["Head"] becomes P[X=1]P[X=1]P[X=1].
  • P["Head"∩"Head"∩"Head"]P[\text{"Head"} \cap \text{"Head"} \cap \text{"Head"}]P["Head"∩"Head"∩"Head"] becomes P[X=3]P[X=3]P[X=3].

This numerical translation streamlines both notation and subsequent calculations for concepts like expectation and variance.

2. Formal Definition of a Random Variable​

A random variable is a function that maps outcomes from a sample space to a numerical value on the real line.

Definition: A random variable XXX is a function X:Ω→RX: \Omega \rightarrow \mathbb{R}X:Ω→R that maps an outcome ξ∈Ω\xi \in \Omegaξ∈Ω from the sample space to a number X(ξ)X(\xi)X(ξ) on the real line R\mathbb{R}R.

  • Sample Space (Ω\OmegaΩ): The set of all possible outcomes of a random experiment. Outcomes ξ\xiξ can be non-numerical or complex.
  • Real Line (R\mathbb{R}R): The set of all real numbers. Random variables transform outcomes into these numerical values.

The primary purpose is to convert qualitative descriptions into quantitative data.

Example: Two Coin Flips​

Consider flipping a fair coin two times. The sample space is Ω={HH, HT, TH, TT}\Omega = \{ \text{HH, HT, TH, TT} \}Ω={HH, HT, TH, TT}.

Let's define a random variable XXX as the number of heads observed.

  • For the outcome ξ1=HH\xi_1 = \text{HH}ξ1​=HH, X(ξ1)=X(HH)=2X(\xi_1) = X(\text{HH}) = 2X(ξ1​)=X(HH)=2.
  • For the outcome ξ2=HT\xi_2 = \text{HT}ξ2​=HT, X(ξ2)=X(HT)=1X(\xi_2) = X(\text{HT}) = 1X(ξ2​)=X(HT)=1.
  • For the outcome ξ3=TH\xi_3 = \text{TH}ξ3​=TH, X(ξ3)=X(TH)=1X(\xi_3) = X(\text{TH}) = 1X(ξ3​)=X(TH)=1.
  • For the outcome ξ4=TT\xi_4 = \text{TT}ξ4​=TT, X(ξ4)=X(TT)=0X(\xi_4) = X(\text{TT}) = 0X(ξ4​)=X(TT)=0.

Here, XXX effectively translates textual outcomes into numerical counts.

3. Calculating Probabilities for Random Variables: The Inverse Mapping​

When we seek to calculate the probability P[X=a]P[X=a]P[X=a] (i.e., the probability that the random variable XXX takes a specific numerical value aaa), we are referring to an event in the numerical domain. However, standard probability measures PPP are defined over events within the original sample space Ω\OmegaΩ. To bridge this, we use the concept of an inverse image.

The event {X=a}\{X=a\}{X=a} signifies that the random variable XXX evaluates to aaa. To find its probability, we must determine which outcomes in Ω\OmegaΩ correspond to this condition.

The inverse image of aaa, denoted X−1(a)X^{-1}(a)X−1(a), is the set of all outcomes ξ∈Ω\xi \in \Omegaξ∈Ω such that X(ξ)=aX(\xi) = aX(ξ)=a. X−1(a)={ξ∈Ω∣X(ξ)=a}X^{-1}(a) = \{ \xi \in \Omega \mid X(\xi) = a \}X−1(a)={ξ∈Ω∣X(ξ)=a} Crucially, this set X−1(a)X^{-1}(a)X−1(a) is an event in the sample space Ω\OmegaΩ. Since it's an event within Ω\OmegaΩ, its probability can be directly measured by the probability measure PPP.

Thus, the probability P[X=a]P[X=a]P[X=a] is calculated as: P[X=a]=P(X−1(a))=P({ξ∈Ω∣X(ξ)=a})P[X=a] = P(X^{-1}(a)) = P(\{ \xi \in \Omega \mid X(\xi) = a \})P[X=a]=P(X−1(a))=P({ξ∈Ω∣X(ξ)=a})

    • X(HT)=1X(\text{HT}) = 1X(HT)=1

  3. Calculate the probability: P[X=1]=P({HT, TH})=P(HT)+P(TH)=14+14=24=12P[X=1] = P(\{\text{HT, TH}\}) = P(\text{HT}) + P(\text{TH}) = \frac{1}{4} + \frac{1}{4} = \frac{2}{4} = \frac{1}{2}P[X=1]=P({HT, TH})=P(HT)+P(TH)=41​+41​=42​=21​

Quick Summary​

  • A random variable is a function X:Ω→RX: \Omega \rightarrow \mathbb{R}X:Ω→R that maps outcomes from a sample space to numerical values, providing a mathematical framework for qualitative events.
  • To calculate the probability P[X=a]P[X=a]P[X=a] (that XXX takes a specific value aaa), we first determine the inverse image X−1(a)X^{-1}(a)X−1(a). This set comprises all original sample space outcomes ξ\xiξ that map to aaa.
  • The set X−1(a)X^{-1}(a)X−1(a) is an event in Ω\OmegaΩ, allowing its probability to be measured using the defined probability measure PPP.
  • In practical applications, while the inverse mapping is fundamental to the definition, we often work directly with P[X=a]P[X=a]P[X=a] once the random variable's behavior is established, without explicitly detailing each inverse mapping step.
