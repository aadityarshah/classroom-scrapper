Cumulative Distribution Functions (Discrete Case)

1. Introduction: Generating Arbitrary Random Numbers​

A fundamental problem in probability and computation is how to generate random numbers following an arbitrary probability mass function (PMF), pX(k)p_X(k)pX​(k). While computers can typically generate uniformly distributed random numbers (e.g., in [0,1][0,1][0,1]), direct generation from an arbitrary discrete PMF is not straightforward.

Consider a PMF pX(k)=[0.1,0.4,0.2,0.3]p_X(k) = [0.1, 0.4, 0.2, 0.3]pX​(k)=[0.1,0.4,0.2,0.3] for k=[1,2,3,4]k = [1, 2, 3, 4]k=[1,2,3,4]. The "trick" to generate samples from this distribution using a uniform random number generator lies in computing the cumulative sum.

By mapping segments of the [0,1][0,1][0,1] interval to the possible outcomes based on their cumulative probabilities, we can simulate the desired distribution. If a uniformly distributed random number U∈[0,1)U \in [0,1)U∈[0,1) falls into a certain segment, we output the corresponding kkk.

2. Cumulative Distribution Function (CDF) Definition​

The Cumulative Distribution Function (CDF), denoted FX(x)F_X(x)FX​(x), for a discrete random variable XXX is defined as the probability that XXX takes a value less than or equal to xxx:

FX(x)≜P[X≤x]=∑x′≤xpX(x′)F_X(x) \triangleq P[X \le x] = \sum_{x' \le x} p_X(x')FX​(x)≜P[X≤x]=x′≤x∑​pX​(x′)

Interpretation:

  • The CDF is conceptually an "integration" or cumulative sum of the PMF.
  • The CDF is always well-defined, unlike the PMF which is typically defined only at discrete points for discrete variables.
  • The CDF framework works universally for both discrete and continuous random variables, providing a unified view of probability distributions.

3. Examples of CDF for Discrete Random Variables​

Let's illustrate the CDF with examples.

Example 1: Simple PMF Consider a random variable XXX with the following PMF: pX(0)=14p_X(0) = \frac{1}{4}pX​(0)=41​, pX(1)=12p_X(1) = \frac{1}{2}pX​(1)=21​, pX(4)=14p_X(4) = \frac{1}{4}pX​(4)=41​. The corresponding CDF FX(x)F_X(x)FX​(x) would be a step function:

  • For x<0x < 0x<0, FX(x)=0F_X(x) = 0FX​(x)=0.
  • For 0≤x<10 \le x < 10≤x<1, FX(x)=P(X≤0)=pX(0)=14F_X(x) = P(X \le 0) = p_X(0) = \frac{1}{4}FX​(x)=P(X≤0)=pX​(0)=41​.
  • For 1≤x<41 \le x < 41≤x<4, FX(x)=P(X≤1)=pX(0)+pX(1)=14+12=34F_X(x) = P(X \le 1) = p_X(0) + p_X(1) = \frac{1}{4} + \frac{1}{2} = \frac{3}{4}FX​(x)=P(X≤1)=pX​(0)+pX​(1)=41​+21​=43​.
  • For x≥4x \ge 4x≥4, FX(x)=P(X≤4)=pX(0)+pX(1)+pX(4)=14+12+14=1F_X(x) = P(X \le 4) = p_X(0) + p_X(1) + p_X(4) = \frac{1}{4} + \frac{1}{2} + \frac{1}{4} = 1FX​(x)=P(X≤4)=pX​(0)+pX​(1)+pX​(4)=41​+21​+41​=1.

This forms a "staircase" function, where the steps occur at the values XXX can take, and the height of each step corresponds to the probability of that value.

Example 2: English Letter Frequencies If pX(k)p_X(k)pX​(k) represents the frequency of English letters (a-z), the CDF FX(k)F_X(k)FX​(k) would show the cumulative probability of observing a letter up to kkk. This also results in a staircase function, with each step representing the probability of a specific letter.

4. Properties of CDFs (Discrete Case)​

For any cumulative distribution function FX(x)F_X(x)FX​(x) of a discrete random variable XXX:

  1. Monotonically Non-decreasing: If x1<x2x_1 < x_2x1​<x2​, then FX(x1)≤FX(x2)F_X(x_1) \le F_X(x_2)FX​(x1​)≤FX​(x2​). The cumulative probability can never decrease.
  2. Limits:
    • lim⁡x→−∞FX(x)=0\lim_{x \to -\infty} F_X(x) = 0limx→−∞​FX​(x)=0
    • lim⁡x→+∞FX(x)=1\lim_{x \to +\infty} F_X(x) = 1limx→+∞​FX​(x)=1
  3. Right-Continuity: The CDF is always continuous from the right. That is, lim⁡ϵ→0+FX(x+ϵ)=FX(x)\lim_{\epsilon \to 0^+} F_X(x+\epsilon) = F_X(x)limϵ→0+​FX​(x+ϵ)=FX​(x). For discrete CDFs, this implies that the solid dot on a graph of the CDF is always on the right side of the jump.
  4. Jumps at Probable Values: At any position xkx_kxk​ where pX(xk)>0p_X(x_k) > 0pX​(xk​)>0, the CDF will have a jump.
  5. Height of Each Jump: The height of a jump at xkx_kxk​ is equal to the probability pX(xk)p_X(x_k)pX​(xk​). Specifically, P(X=xk)=FX(xk)−FX(xk−)P(X=x_k) = F_X(x_k) - F_X(x_k^-)P(X=xk​)=FX​(xk​)−FX​(xk−​), where FX(xk−)F_X(x_k^-)FX​(xk−​) is the limit of FX(x)F_X(x)FX​(x) as xxx approaches xkx_kxk​ from the left.

5. Converting Between PMF and CDF​

For a discrete random variable XXX, the PMF can be obtained from its CDF. If XXX can take values x1,x2,…,xNx_1, x_2, \dots, x_Nx1​,x2​,…,xN​ in increasing order, then:

pX(xk)=FX(xk)−FX(xk−1)for k=1,…,Np_X(x_k) = F_X(x_k) - F_X(x_{k-1}) \quad \text{for } k=1, \dots, NpX​(xk​)=FX​(xk​)−FX​(xk−1​)for k=1,…,N

For x0x_0x0​ (the value just before the smallest possible value x1x_1x1​), we define FX(x0)=FX(−∞)=0F_X(x_0) = F_X(-\infty) = 0FX​(x0​)=FX​(−∞)=0.

A simpler version for integer-valued random variables:

pX(k)=FX(k)−FX(k−1)p_X(k) = F_X(k) - F_X(k-1)pX​(k)=FX​(k)−FX​(k−1)

Example: Finding PMF from CDF Given the CDF: FX(0)=14F_X(0)=\frac{1}{4}FX​(0)=41​, FX(1)=34F_X(1)=\frac{3}{4}FX​(1)=43​, FX(4)=1F_X(4)=1FX​(4)=1. We know the PMF will have non-negative values only at x=0,1,4x=0, 1, 4x=0,1,4.

  • pX(0)=FX(0)−FX(−∞)=14−0=14p_X(0) = F_X(0) - F_X(-\infty) = \frac{1}{4} - 0 = \frac{1}{4}pX​(0)=FX​(0)−FX​(−∞)=41​−0=41​.
  • pX(1)=FX(1)−FX(0)=34−14=24=12p_X(1) = F_X(1) - F_X(0) = \frac{3}{4} - \frac{1}{4} = \frac{2}{4} = \frac{1}{2}pX​(1)=FX​(1)−FX​(0)=43​−41​=42​=21​.
  • pX(4)=FX(4)−FX(1)=1−34=14p_X(4) = F_X(4) - F_X(1) = 1 - \frac{3}{4} = \frac{1}{4}pX​(4)=FX​(4)−FX​(1)=1−43​=41​.

This successfully recovers the PMF from Example 1.

6. Expressing PMF as Delta Functions (Advanced View)​

A PMF for a discrete random variable can be formally expressed using Dirac delta functions. This perspective clarifies why the PMF is not a "function" in the traditional sense of mapping real numbers to probabilities everywhere, while the CDF is.

pX(x)=∑k∈X(Ω)pX(k)⋅δ(x−k)p_X(x) = \sum_{k \in X(\Omega)} p_X(k) \cdot \delta(x-k)pX​(x)=k∈X(Ω)∑​pX​(k)⋅δ(x−k)

Here, δ(x−k)\delta(x-k)δ(x−k) is the Dirac delta function, which is zero everywhere except at x=kx=kx=k, where it is infinitely high such that its integral is 1.

For instance, using the PMF from Example 1:

pX(x)=14δ(x)+12δ(x−1)+14δ(x−4)p_X(x) = \frac{1}{4}\delta(x) + \frac{1}{2}\delta(x-1) + \frac{1}{4}\delta(x-4)pX​(x)=41​δ(x)+21​δ(x−1)+41​δ(x−4)

Integrating this gives the CDF in terms of the Heaviside step function u(x)u(x)u(x):

FX(x)=14u(x)+12u(x−1)+14u(x−4)F_X(x) = \frac{1}{4}u(x) + \frac{1}{2}u(x-1) + \frac{1}{4}u(x-4)FX​(x)=41​u(x)+21​u(x−1)+41​u(x−4)

This further emphasizes that pX(x)p_X(x)pX​(x) (as a sum of delta functions) is a generalized function (or distribution), while FX(x)F_X(x)FX​(x) (as a sum of step functions) is a well-behaved function.

Quick Summary​

  • The Cumulative Distribution Function (CDF), FX(x)=P[X≤x]F_X(x) = P[X \le x]FX​(x)=P[X≤x], is the cumulative sum of the Probability Mass Function (PMF) for discrete random variables.
  • CDFs are always defined and serve as a unified way to describe probability distributions for both discrete and continuous random variables.
  • For discrete random variables, CDFs are staircase functions, where jumps occur at the values the random variable can take.
  • Key properties of CDFs include being non-decreasing, having limits of 0 at −∞-\infty−∞ and 1 at +∞+\infty+∞, and being right-continuous.
  • The PMF can be derived from the CDF by pX(k)=FX(k)−FX(k−1)p_X(k) = F_X(k) - F_X(k-1)pX​(k)=FX​(k)−FX​(k−1) for integer-valued kkk.
  • CDFs are crucial for generating random numbers from arbitrary discrete distributions using uniform random number generators.
  • We will revisit CDFs extensively when discussing continuous random variables.
