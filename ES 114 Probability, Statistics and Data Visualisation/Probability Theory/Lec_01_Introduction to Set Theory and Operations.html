Set Theory Fundamentals

In the realm of probability and stochastic processes, a solid understanding of set theory is absolutely fundamental. Sets provide the language and framework for defining sample spaces, events, and relationships between them. This lecture lays the groundwork by introducing core concepts of set theory and their associated operations.

1. Sets and Elements​

Definition of a Set​

A set is a well-defined collection of distinct objects. These objects are referred to as elements or members of the set. We typically denote a set with a capital letter, e.g., AAA, and its elements are enclosed in curly braces. For example, A={ξ1,ξ2,…,ξn}A = \{\xi_1, \xi_2, \dots, \xi_n\}A={ξ1​,ξ2​,…,ξn​} represents a set where ξi\xi_iξi​ is the iii-th element.

Set Membership​

The relationships between an element and a set are denoted as follows:

  • ξ∈A\xi \in Aξ∈A: An object ξ\xiξ is an element of set AAA.
  • ξ∉A\xi \notin Aξ∈/A: An object ξ\xiξ is not an element of set AAA.

Types of Sets​

Sets can be categorized based on the number and nature of their elements:

  • Finite Set: A set containing a limited or fixed number of elements.
    • Example: A={0,1}A = \{0, 1\}A={0,1}.
  • Countable Set: A set whose elements can be put into a one-to-one correspondence with the set of natural numbers N={1,2,3,… }\mathbb{N} = \{1, 2, 3, \dots\}N={1,2,3,…}. This includes finite sets and infinite sets that can be "counted" (enumerated).
    • Example: A={2,4,6,8,… }A = \{2, 4, 6, 8, \dots\}A={2,4,6,8,…} (the set of positive even integers).
  • Uncountable Set: A set whose elements cannot be put into a one-to-one correspondence with the natural numbers. These sets are "larger" than countable sets.
    • Example: A={x∣0<x<1}A = \{x \mid 0 < x < 1\}A={x∣0<x<1} (the set of real numbers between 0 and 1).

Open and Closed Intervals​

For sets of real numbers, intervals are commonly used:

  • Open Interval: (a,b)={x∣a<x<b}(a, b) = \{x \mid a < x < b\}(a,b)={x∣a<x<b}. This set includes all real numbers strictly between aaa and bbb, but does not include aaa or bbb.
  • Closed Interval: [a,b]={x∣a≤x≤b}[a, b] = \{x \mid a \le x \le b\}[a,b]={x∣a≤x≤b}. This set includes all real numbers between aaa and bbb, including aaa and bbb.

Sets of Functions​

Elements of a set are not limited to numbers; they can be any type of object, including functions. For instance, one could define a set of all straight lines or a set of all periodic cosine functions.

2. Subsets​

Definition of a Subset​

A set BBB is a subset of AAA, denoted B⊆AB \subseteq AB⊆A, if every element of BBB is also an element of AAA. In other words, for any ξ∈B\xi \in Bξ∈B, it must also be true that ξ∈A\xi \in Aξ∈A.

Proper and Improper Subsets​

  • Proper Subset: If B⊆AB \subseteq AB⊆A and B≠AB \ne AB=A, then BBB is a proper subset of AAA, denoted B⊂AB \subset AB⊂A. This means AAA contains at least one element not found in BBB.
    • Example: If A={1,2,3,4,5,6}A = \{1, 2, 3, 4, 5, 6\}A={1,2,3,4,5,6} and B={1,2}B = \{1, 2\}B={1,2}, then BBB is a proper subset of AAA.
  • Improper Subset: If B⊆AB \subseteq AB⊆A and B=AB = AB=A, then BBB is an improper subset of AAA. In this case, the sets are identical.
    • Example: If A={1,2,3}A = \{1, 2, 3\}A={1,2,3} and B={1,2,3}B = \{1, 2, 3\}B={1,2,3}, then BBB is an improper subset of AAA.

Theorem: Equality of Sets​

Theorem: If A⊆BA \subseteq BA⊆B and B⊆AB \subseteq AB⊆A, then A=BA = BA=B. Proof Sketch: Assume, for the sake of contradiction, that A≠BA \ne BA=B. If A⊆BA \subseteq BA⊆B and B⊆AB \subseteq AB⊆A, but A≠BA \ne BA=B, it would imply that there is an element in AAA not in BBB, or an element in BBB not in AAA.

  1. If there exists an x∈Ax \in Ax∈A such that x∉Bx \notin Bx∈/B, this contradicts the condition A⊆BA \subseteq BA⊆B.
  2. If there exists an x∈Bx \in Bx∈B such that x∉Ax \notin Ax∈/A, this contradicts the condition B⊆AB \subseteq AB⊆A. Since both scenarios lead to a contradiction, the initial assumption that A≠BA \ne BA=B must be false. Therefore, A=BA = BA=B.

3. Empty and Universal Sets​

Empty Set​

  • Definition: The empty set, denoted ∅\emptyset∅ or {}\{\}{}, is a unique set that contains no elements.
  • Property: The empty set is a subset of every set. That is, for any set AAA, ∅⊆A\emptyset \subseteq A∅⊆A.

Universal Set​

  • Definition: The universal set, denoted Ω\OmegaΩ (or SSS for sample space in probability), is the set containing all possible elements relevant to a particular context or problem.
  • Property: Every set AAA under consideration in that context is a subset of the universal set, i.e., A⊆ΩA \subseteq \OmegaA⊆Ω. The universal set is also a subset of itself.

4. Set Operations: Union​

Finite Union​

  • Definition: The union of two sets AAA and BBB, denoted A∪BA \cup BA∪B, is the set containing all elements that are in AAA or in BBB (or both). A∪B={ξ∣ξ∈A or ξ∈B}A \cup B = \{\xi \mid \xi \in A \text{ or } \xi \in B\}A∪B={ξ∣ξ∈A or ξ∈B}
  • Example 1: If A={1,2,3,4}A = \{1, 2, 3, 4\}A={1,2,3,4} and B={1,5,6}B = \{1, 5, 6\}B={1,5,6}, then A∪B={1,2,3,4,5,6}A \cup B = \{1, 2, 3, 4, 5, 6\}A∪B={1,2,3,4,5,6}.
  • Example 2: If A={t∣3<t<4}A = \{t \mid 3 < t < 4\}A={t∣3<t<4} and B={t∣t≥3.5}B = \{t \mid t \ge 3.5\}B={t∣t≥3.5}, then A∪B={t∣t>3}A \cup B = \{t \mid t > 3\}A∪B={t∣t>3}.

Infinite Union​

  • Definition: The infinite union of a sequence of sets A1,A2,…,An,…A_1, A_2, \dots, A_n, \dotsA1​,A2​,…,An​,… is denoted ⋃n=1∞An\bigcup_{n=1}^\infty A_n⋃n=1∞​An​. An element xxx belongs to this union if xxx is in at least one of the sets AnA_nAn​. ⋃n=1∞An={x∣∃n∈N,x∈An}\bigcup_{n=1}^\infty A_n = \{x \mid \exists n \in \mathbb{N}, x \in A_n \}⋃n=1∞​An​={x∣∃n∈N,x∈An​}
  • Example: Consider the sequence of sets An=[0,1−1/n]A_n = [0, 1 - 1/n]An​=[0,1−1/n] for n∈Nn \in \mathbb{N}n∈N.
    • A1=[0,0]A_1 = [0, 0]A1​=[0,0]
    • A2=[0,0.5]A_2 = [0, 0.5]A2​=[0,0.5]
    • A3=[0,2/3]A_3 = [0, 2/3]A3​=[0,2/3]
    • As n→∞n \to \inftyn→∞, the upper bound 1−1/n1 - 1/n1−1/n approaches 111. The union accumulates all points up to, but not including, 111.
    • Thus, ⋃n=1∞An=[0,1)\bigcup_{n=1}^\infty A_n = [0, 1)⋃n=1∞​An​=[0,1).

5. Set Operations: Intersection​

Finite Intersection​

  • Definition: The intersection of two sets AAA and BBB, denoted A∩BA \cap BA∩B, is the set containing all elements that are in AAA and in BBB. A∩B={ξ∣ξ∈A and ξ∈B}A \cap B = \{\xi \mid \xi \in A \text{ and } \xi \in B\}A∩B={ξ∣ξ∈A and ξ∈B}
  • Example 1: If A={1,2,3,4}A = \{1, 2, 3, 4\}A={1,2,3,4} and B={1,5,6}B = \{1, 5, 6\}B={1,5,6}, then A∩B={1}A \cap B = \{1\}A∩B={1}.
  • Example 2: If A={t∣3<t<4}A = \{t \mid 3 < t < 4\}A={t∣3<t<4} and B={t∣t≥3.5}B = \{t \mid t \ge 3.5\}B={t∣t≥3.5}, then A∩B={t∣3.5≤t<4}A \cap B = \{t \mid 3.5 \le t < 4\}A∩B={t∣3.5≤t<4}.

Infinite Intersection​

  • Definition: The infinite intersection of a sequence of sets A1,A2,…,An,…A_1, A_2, \dots, A_n, \dotsA1​,A2​,…,An​,… is denoted ⋂n=1∞An\bigcap_{n=1}^\infty A_n⋂n=1∞​An​. An element xxx belongs to this intersection if xxx is in all of the sets AnA_nAn​. ⋂n=1∞An={x∣∀n∈N,x∈An}\bigcap_{n=1}^\infty A_n = \{x \mid \forall n \in \mathbb{N}, x \in A_n \}⋂n=1∞​An​={x∣∀n∈N,x∈An​}
  • Example: Consider the sequence of sets An=[0,1+1/n]A_n = [0, 1 + 1/n]An​=[0,1+1/n] for n∈Nn \in \mathbb{N}n∈N.
    • A1=[0,2]A_1 = [0, 2]A1​=[0,2]
    • A2=[0,1.5]A_2 = [0, 1.5]A2​=[0,1.5]
    • A3=[0,4/3]A_3 = [0, 4/3]A3​=[0,4/3]
    • As n→∞n \to \inftyn→∞, the upper bound 1+1/n1 + 1/n1+1/n approaches 111. The intersection must contain only elements that are common to all these sets. The common interval shrinks to include only elements from 000 up to 111.
    • Thus, ⋂n=1∞An=[0,1]\bigcap_{n=1}^\infty A_n = [0, 1]⋂n=1∞​An​=[0,1].

6. Set Operations: Complement and Difference​

Complement​

  • Definition: The complement of a set AAA, denoted AcA^cAc or A′A'A′, is the set of all elements in the universal set Ω\OmegaΩ that are not in AAA. Ac={ξ∣ξ∈Ω and ξ∉A}A^c = \{\xi \mid \xi \in \Omega \text{ and } \xi \notin A\}Ac={ξ∣ξ∈Ω and ξ∈/A}
  • Example: Let Ω={integers}\Omega = \{\text{integers}\}Ω={integers} and A={even integers}A = \{\text{even integers}\}A={even integers}. Then Ac={odd integers}A^c = \{\text{odd integers}\}Ac={odd integers}.

Set Difference​

  • Definition: The difference of set AAA and set BBB, denoted A∖BA \setminus BA∖B or A−BA - BA−B, is the set of all elements that are in AAA but not in BBB. A∖B={ξ∣ξ∈A and ξ∉B}A \setminus B = \{\xi \mid \xi \in A \text{ and } \xi \notin B\}A∖B={ξ∣ξ∈A and ξ∈/B}
  • It can also be expressed in terms of intersection and complement: A∖B=A∩BcA \setminus B = A \cap B^cA∖B=A∩Bc.

7. Disjoint Sets and Partitions​

Disjoint Sets​

  • Definition: Two sets AAA and BBB are said to be disjoint (or mutually exclusive) if their intersection is the empty set, i.e., A∩B=∅A \cap B = \emptysetA∩B=∅. They share no common elements.
  • For a collection of sets: A collection of sets {A1,A2,…,An}\{A_1, A_2, \dots, A_n\}{A1​,A2​,…,An​} is said to be disjoint (or mutually exclusive) if Ai∩Aj=∅A_i \cap A_j = \emptysetAi​∩Aj​=∅ for all i≠ji \ne ji=j.

Partition​

  • Definition: A collection of sets {A1,A2,…,An}\{A_1, A_2, \dots, A_n\}{A1​,A2​,…,An​} forms a partition of the universal set Ω\OmegaΩ if it satisfies two conditions:
    1. Non-overlap (Disjoint): The sets are mutually exclusive, meaning Ai∩Aj=∅A_i \cap A_j = \emptysetAi​∩Aj​=∅ for all i≠ji \ne ji=j.
    2. Decomposition (Completeness): The union of all sets equals the universal set, meaning A1∪A2∪⋯∪An=ΩA_1 \cup A_2 \cup \dots \cup A_n = \OmegaA1​∪A2​∪⋯∪An​=Ω.

8. Laws of Set Operations​

These fundamental laws govern how set operations interact and are crucial for simplifying and manipulating set expressions.

Commutative Laws​

The order of sets in union or intersection does not affect the result.

  • A∩B=B∩AA \cap B = B \cap AA∩B=B∩A
  • A∪B=B∪AA \cup B = B \cup AA∪B=B∪A

Associative Laws​

For three sets, the grouping of sets in union or intersection does not affect the result.

  • A∪(B∪C)=(A∪B)∪CA \cup (B \cup C) = (A \cup B) \cup CA∪(B∪C)=(A∪B)∪C
  • A∩(B∩C)=(A∩B)∩CA \cap (B \cap C) = (A \cap B) \cap CA∩(B∩C)=(A∩B)∩C

Distributive Laws​

These laws describe how union and intersection distribute over each other.

  • A∩(B∪C)=(A∩B)∪(A∩C)A \cap (B \cup C) = (A \cap B) \cup (A \cap C)A∩(B∪C)=(A∩B)∪(A∩C)
  • A∪(B∩C)=(A∪B)∩(A∪C)A \cup (B \cap C) = (A \cup B) \cap (A \cup C)A∪(B∩C)=(A∪B)∩(A∪C)

De Morgan's Laws​

These laws relate complements of unions and intersections, providing a way to transform one into the other.

  • (A∩B)c=Ac∪Bc(A \cap B)^c = A^c \cup B^c(A∩B)c=Ac∪Bc (The complement of an intersection is the union of the complements)
  • (A∪B)c=Ac∩Bc(A \cup B)^c = A^c \cap B^c(A∪B)c=Ac∩Bc (The complement of a union is the intersection of the complements)

Quick Summary​

This lecture provided a foundational introduction to set theory, a prerequisite for understanding probability. We defined what a set is, distinguished between finite, countable, and uncountable sets, and clarified set membership. Key concepts included subsets (proper and improper), the empty set, and the universal set. We then thoroughly explored fundamental set operations: union (finite and infinite), intersection (finite and infinite), complement, and set difference. Finally, we defined disjoint sets and partitions, concluding with a review of essential set operation laws, namely commutative, associative, distributive, and De Morgan's laws. These principles form the bedrock for constructing more complex probabilistic models.
