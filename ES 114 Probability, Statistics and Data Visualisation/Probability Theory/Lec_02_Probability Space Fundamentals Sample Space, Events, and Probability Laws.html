Probability Space

Welcome to Lecture 2 of ECE 302. Today, we embark on a journey into the fundamental building blocks of probability theory: the Probability Space. Understanding these core concepts is paramount for anyone delving into the fascinating world of uncertainty and random phenomena.

What is Probability?​

At its most basic, probability is:

  • A number.
  • Always between 000 and 111, inclusive (0≤P(A)≤10 \le P(A) \le 10≤P(A)≤1).
  • Always associated with an event.

For instance, consider the probability of getting a Head when tossing a fair coin. We denote this as P("H")P(\text{"H"})P("H"). If the coin is fair, P("H")=0.5P(\text{"H"}) = 0.5P("H")=0.5.

Three Elements of a Probability Model​

A complete probability model, often called a probability space, is constructed from three fundamental elements:

  1. Sample Space (Ω\OmegaΩ): The set of all possible outcomes of a random experiment.
  2. Event Space (F\mathcal{F}F): A collection of subsets of the sample space, where each subset is called an event, and for which we can assign a probability.
  3. Probability Law (PPP): A function that assigns a probability to each event in the event space.

Let's dissect each of these elements in detail.

The Sample Space (Ω\OmegaΩ)​

Definition​

The sample space, denoted by Ω\OmegaΩ (uppercase Omega), is the collection of all possible outcomes of a random experiment. Each individual outcome within Ω\OmegaΩ is typically denoted by ω\omegaω (lowercase omega).

Examples​

  1. Coin Flip:

    • Ω={Head (H),Tail (T)}\Omega = \{\text{Head (H)}, \text{Tail (T)}\}Ω={Head (H),Tail (T)}

  2. Throw a Dice:

    • Ω={1,2,3,4,5,6}\Omega = \{1, 2, 3, 4, 5, 6\}Ω={1,2,3,4,5,6}

  3. Waiting Time for a Bus: (in minutes, assuming a maximum wait of 30 minutes)

    • Ω={t∈R∣0≤t≤30}\Omega = \{t \in \mathbb{R} \mid 0 \le t \le 30\}Ω={t∈R∣0≤t≤30}
    • This is a continuous sample space.

Elements in the Sample Space​

The nature of the elements in a sample space can vary widely:

  • Discrete Numbers: Such as {1,2,3,…,N}\{1, 2, 3, \ldots, N\}{1,2,3,…,N}.
  • Continuous Intervals: For example, [a,b][a, b][a,b] on the real number line.
  • Functions: In advanced stochastic processes, outcomes can be entire functions or sequences.

The Event Space (F\mathcal{F}F)​

Definition (Event)​

An event, denoted by FFF, is a subset of the sample space Ω\OmegaΩ. Events are specific collections of outcomes for which we want to determine a probability.

Outcome vs. Event​

An outcome ω\omegaω is a single result of the experiment. An event FFF is a set of outcomes. For example, in a dice roll, 'getting a 3' is an outcome. 'Getting an even number' is an event composed of multiple outcomes.

Examples​

  1. Throw a Dice: Let Ω={1,2,3,4,5,6}\Omega = \{1, 2, 3, 4, 5, 6\}Ω={1,2,3,4,5,6}.

    • F1={even numbers}={2,4,6}F_1 = \{\text{even numbers}\} = \{2, 4, 6\}F1​={even numbers}={2,4,6}
    • F2={less than 3}={1,2}F_2 = \{\text{less than } 3\} = \{1, 2\}F2​={less than 3}={1,2}

  2. Wait a Bus: Let Ω={t∈R∣0≤t≤30}\Omega = \{t \in \mathbb{R} \mid 0 \le t \le 30\}Ω={t∈R∣0≤t≤30}.

    • F1={0<t<10}F_1 = \{0 < t < 10\}F1​={0<t<10} (waiting time is less than 10 minutes)
    • F2={0<t<5}∪{20<t≤30}F_2 = \{0 < t < 5\} \cup \{20 < t \le 30\}F2​={0<t<5}∪{20<t≤30} (waiting time is less than 5 minutes or between 20 and 30 minutes)

Definition (Event Space)​

The collection of all events for a given sample space is called the event space, denoted as F\mathcal{F}F. It is a set of subsets of Ω\OmegaΩ.

How Many Events?​

Question: If you have nnn elements in the sample space Ω\OmegaΩ, how many distinct events can you construct?

Solution: Each element ω∈Ω\omega \in \Omegaω∈Ω can either be included in an event or not. Since there are nnn elements, and each has two choices, the total number of possible subsets (events) is 2n2^n2n. This includes the empty set ∅\emptyset∅ (the impossible event) and the sample space Ω\OmegaΩ itself (the sure event).

Sigma-Field (Optional)​

The event space is often referred to as a σ\sigmaσ-field (or σ\sigmaσ-algebra) to emphasize its structural properties. A collection of subsets F\mathcal{F}F of Ω\OmegaΩ is a σ\sigmaσ-field if it satisfies the following two properties:

  1. Closure under Complementation: If an event F∈FF \in \mathcal{F}F∈F, then its complement Fc=Ω∖FF^c = \Omega \setminus FFc=Ω∖F must also be in F\mathcal{F}F.
  2. Closure under Countable Unions: If F1,F2,…F_1, F_2, \ldotsF1​,F2​,… is a countable sequence of events in F\mathcal{F}F, then their union ⋃i=1∞Fi\bigcup_{i=1}^\infty F_i⋃i=1∞​Fi​ must also be in F\mathcal{F}F. (Note: This property implies closure under countable intersections as well, by De Morgan's laws: (⋃Fi)c=⋂Fic(\bigcup F_i)^c = \bigcap F_i^c(⋃Fi​)c=⋂Fic​).

Example: For Ω={H,T}\Omega = \{\text{H}, \text{T}\}Ω={H,T}, the σ\sigmaσ-field is: F={∅,{H},{T},{H,T}}\mathcal{F} = \{\emptyset, \{\text{H}\}, \{\text{T}\}, \{\text{H}, \text{T}\}\}F={∅,{H},{T},{H,T}}. Here, n=2n=2n=2, so 2n=22=42^n = 2^2 = 42n=22=4 events. This example perfectly illustrates a σ\sigmaσ-field for a finite sample space.

The Probability Law (PPP)​

Definition​

A probability law (or probability measure), denoted by PPP, is a function that maps an event A∈FA \in \mathcal{F}A∈F to a real number in the interval [0,1][0, 1][0,1]. P:F→[0,1]P: \mathcal{F} \to [0, 1]P:F→[0,1] The probability law quantifies the likelihood of an event occurring.

Example: Flipping a Coin​

Consider flipping a coin. The sample space is Ω={H,T}\Omega = \{\text{H}, \text{T}\}Ω={H,T}. The event space is F={∅,{H},{T},Ω}\mathcal{F} = \{\emptyset, \{\text{H}\}, \{\text{T}\}, \Omega\}F={∅,{H},{T},Ω}.

A sensible probability law for a fair coin would be:

  • P(∅)=0P(\emptyset) = 0P(∅)=0 (probability of no outcome is 0)
  • P({H})=0.5P(\{\text{H}\}) = 0.5P({H})=0.5
  • P({T})=0.5P(\{\text{T}\}) = 0.5P({T})=0.5
  • P(Ω)=P({H,T})=1P(\Omega) = P(\{\text{H}, \text{T}\}) = 1P(Ω)=P({H,T})=1 (probability of getting any outcome is 1)

Probability Law as a Measure (Optional)​

In more advanced contexts, the probability law is often seen as a specific type of measure. A measure, in general, assigns a "size" or "quantity" to subsets of a set. For probability, this "size" is normalized to be between 0 and 1.

Probability can be thought of as the relative size of an event (set) with respect to the sample space. P[E]=Size of ESize of ΩP[E] = \frac{\text{Size of } E}{\text{Size of } \Omega}P[E]=Size of ΩSize of E​

Examples of "Size"​

  • Discrete Numbers: For discrete sample spaces, the "size" of an event is often the count of outcomes it contains. For example, if Ω={1,2,3,4,5,6}\Omega = \{1,2,3,4,5,6\}Ω={1,2,3,4,5,6} and E1={2,4,6}E_1 = \{2,4,6\}E1​={2,4,6}, then Size(E1)=3\text{Size}(E_1) = 3Size(E1​)=3 and Size(Ω)=6\text{Size}(\Omega) = 6Size(Ω)=6. So, P[E1]=3/6=0.5P[E_1] = 3/6 = 0.5P[E1​]=3/6=0.5.
  • 1D Intervals: For continuous sample spaces on a line, the "size" is often the length of the interval. If Ω=[0,10]\Omega = [0, 10]Ω=[0,10] and E1=[1,4]E_1 = [1, 4]E1​=[1,4], then Size(E1)=3\text{Size}(E_1) = 3Size(E1​)=3 and Size(Ω)=10\text{Size}(\Omega) = 10Size(Ω)=10. So, P[E1]=3/10=0.3P[E_1] = 3/10 = 0.3P[E1​]=3/10=0.3.
  • 2D Sets: For continuous sample spaces in two dimensions, the "size" is often the area of the region.

Measure Zero (Optional)​

An important consequence of thinking about probability as a measure (especially for continuous spaces) is the concept of measure zero.

An isolated point in a continuous interval has zero probability. For example, if Ω=[0,1]\Omega = [0, 1]Ω=[0,1], the probability of observing exactly 0.50.50.5 is P[{0.5}]=0P[\{0.5\}] = 0P[{0.5}]=0. This is because the "length" of a single point is zero.

Examples of Measure Zero Sets:

  1. Let Ω=[0,1]\Omega = [0, 1]Ω=[0,1]. The set {0.5}\{0.5\}{0.5} has measure zero, P[{0.5}]=0P[\{0.5\}] = 0P[{0.5}]=0.
  2. Let Ω={1,2,3,4,5,6}\Omega = \{1, 2, 3, 4, 5, 6\}Ω={1,2,3,4,5,6}. Here, discrete outcomes have non-zero probability. The set {1}\{1\}{1} has a probability of 1/61/61/6, not zero.
  3. For any intervals, P[[a,b]]=P[(a,b)]P[[a, b]] = P[(a, b)]P[[a,b]]=P[(a,b)] because the two end points aaa and bbb have measure zero: P[{a}]=P[{b}]=0P[\{a\}] = P[\{b\}] = 0P[{a}]=P[{b}]=0. This means including or excluding the endpoints of an interval does not change its probability in a continuous distribution.

Almost Surely (a.s.)​

In continuous probability, a concept called "almost surely" is crucial.

Definition: An event A∈FA \in \mathcal{F}A∈F is said to hold almost surely (a.s.) if P[A]=1P[A] = 1P[A]=1, even if the event AAA does not include all outcomes in Ω\OmegaΩ. This means the outcomes not in AAA form a set of measure zero.

For example, if you pick a random real number in [0,1][0,1][0,1], the probability that it is not exactly 0.50.50.5 is 111. So, "the number is not 0.50.50.5" happens almost surely.

The Probability Space​

Finally, we unify these three components.

Definition​

A probability space consists of a triplet (Ω,F,P)(\Omega, \mathcal{F}, P)(Ω,F,P), where:

  1. Ω\OmegaΩ is the sample space, the set of all possible outcomes.
  2. F\mathcal{F}F is the event space (or σ\sigmaσ-field), a collection of subsets of Ω\OmegaΩ called events, which satisfies the properties of a σ\sigmaσ-field.
  3. PPP is the probability law (or probability measure), a function P:F→[0,1]P: \mathcal{F} \to [0, 1]P:F→[0,1] that assigns a probability to each event in F\mathcal{F}F and satisfies the axioms of probability (which we will cover in the next lecture).

This triplet provides the mathematical framework for rigorously defining and analyzing random phenomena. Every problem in probability theory ultimately operates within the context of a defined probability space.

Quick Summary​

This lecture established the three foundational components of a probability model:

  • The sample space (Ω\OmegaΩ) is the complete set of all possible outcomes of an experiment. Outcomes can be discrete numbers, continuous values, or even complex functions.
  • An event (FFF) is any subset of the sample space for which we wish to assign a probability. The collection of all such events forms the event space (F\mathcal{F}F), which mathematically is a σ\sigmaσ-field, ensuring closure under set operations like complementation and countable unions.
  • The probability law (PPP) is a function that maps events from the event space to a value between 000 and 111, quantifying their likelihood. It can be intuitively understood as a measure of the relative "size" of an event within the sample space, leading to concepts like measure zero sets and "almost surely" events for continuous distributions. Together, these three elements form the probability space (Ω,F,P\Omega, \mathcal{F}, PΩ,F,P), the essential framework for all probability calculations.
