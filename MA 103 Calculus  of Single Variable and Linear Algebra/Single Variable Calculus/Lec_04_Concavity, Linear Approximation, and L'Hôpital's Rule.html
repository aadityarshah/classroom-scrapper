Concavity​

DEFINITION: Concave Up, Concave Down The graph of a differentiable function y=f(x)y = f(x)y=f(x) is:

  • Concave Up on an open interval III if f′f'f′ is increasing on III.
  • Concave Down on an open interval III if f′f'f′ is decreasing on III.

Second Derivative Test for Concavity​

THEOREM: Second Derivative Test for Concavity Let y=f(x)y = f(x)y=f(x) be twice-differentiable on an interval III.

  1. If f′′(x)f''(x) f′′(x)>0 00 on III, the graph of fff over III is concave up.
  2. If f′′(x)f''(x) f′′(x)<0 00 on III, the graph of fff over III is concave down.

Point of Inflection​

DEFINITION: Point of Inflection A point where the graph of a function has a tangent line and where the concavity changes is a point of inflection. A point of inflection may occur where f′′(x)=0f''(x) = 0f′′(x)=0 or where f′′(x)f''(x)f′′(x) does not exist, provided the concavity actually changes sign around that point.

Second Derivative Test for Local Extrema​

THEOREM: Second Derivative Test for Local Extrema Suppose f′′(x)f''(x)f′′(x) is continuous on an open interval containing ccc with f′(c)=0f'(c) = 0f′(c)=0.

  • If f′′(c)f''(c) f′′(c)>0 00, then fff has a local minimum at ccc.
  • If f′′(c)f''(c) f′′(c)<0 00, then fff has a local maximum at ccc.
  • If f′′(c)=0f''(c) = 0f′′(c)=0, the test is inconclusive; fff may have a local maximum, a local minimum, or neither at ccc.

Quick Summary​

This lecture clarified how the second derivative dictates a function's concavity (upward or downward) and the identification of inflection points where concavity shifts. The Second Derivative Test was detailed for classifying critical points as local maxima or minima based on the sign of f′′(c)f''(c)f′′(c). Linear approximation, defined as the tangent line L(x)=f(a)+f′(a)(x−a)L(x) = f(a) + f'(a)(x-a)L(x)=f(a)+f′(a)(x−a), was presented as a method for local estimation. L'Hôpital's Rule provided a crucial tool for evaluating indeterminate limits of the 0/00/00/0 or ∞/∞\infty/\infty∞/∞ forms by examining the ratio of derivatives. Finally, the lecture introduced Cauchy's Mean Value Theorem, extending the Mean Value Theorem to two functions, and Darboux's Theorem, which establishes the Intermediate Value Property for derivatives.
