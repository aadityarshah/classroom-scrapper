Gram-Schmidt Orthogonalization​

The Gram-Schmidt (G-S) process is a method to obtain an orthonormal basis from any basis of a finite-dimensional Inner Product Space (IPS).

Theorem (Orthogonal Set Basis): If VVV is an IPS and S={v1,…,vk}S = \{v_1, \dots, v_k\}S={v1​,…,vk​} is an orthogonal set with vi≠0v_i \neq 0vi​=0 for all iii, then for any y∈Span(S)y \in \text{Span}(S)y∈Span(S), y=∑i=1k⟨y,vi⟩∥vi∥2viy = \sum_{i=1}^k \frac{\langle y, v_i \rangle}{\|v_i\|^2} v_iy=∑i=1k​∥vi​∥2⟨y,vi​⟩​vi​ If SSS is orthonormal, then y=∑i=1k⟨y,vi⟩viy = \sum_{i=1}^k \langle y, v_i \rangle v_iy=∑i=1k​⟨y,vi​⟩vi​. An orthogonal set of non-zero vectors is linearly independent.

Theorem (Gram-Schmidt Process): Let VVV be an IPS, and S={w1,…,wn}S = \{w_1, \dots, w_n\}S={w1​,…,wn​} be a linearly independent set of VVV. Define S′={v1,…,vn}S' = \{v_1, \dots, v_n\}S′={v1​,…,vn​} as: v1=w1v_1 = w_1v1​=w1​ vk=wk−∑j=1k−1⟨wk,vj⟩∥vj∥2vjfor 2≤k≤nv_k = w_k - \sum_{j=1}^{k-1} \frac{\langle w_k, v_j \rangle}{\|v_j\|^2} v_j \quad \text{for } 2 \le k \le nvk​=wk​−∑j=1k−1​∥vj​∥2⟨wk​,vj​⟩​vj​for 2≤k≤n Then S′S'S′ is an orthogonal set of nonzero vectors, and Span(S′)=Span(S)\text{Span}(S') = \text{Span}(S)Span(S′)=Span(S). If SSS is a basis for VVV, the G-S method yields an orthonormal basis S′S'S′ for VVV by normalizing the vkv_kvk​ vectors.

Orthogonal Complement​

Definition: For a subset SSS of an IPS VVV, its orthogonal complement S⊥S^\perpS⊥ is defined as: S⊥={x∈V∣⟨x,y⟩=0 for all y∈S}S^\perp = \{x \in V \mid \langle x, y \rangle = 0 \text{ for all } y \in S\}S⊥={x∈V∣⟨x,y⟩=0 for all y∈S}

Theorem (Orthogonal Decomposition): Let WWW be a finite-dimensional subspace of an IPS VVV. For any y∈Vy \in Vy∈V, there exist unique vectors u∈Wu \in Wu∈W and z∈W⊥z \in W^\perpz∈W⊥ such that y=u+zy = u + zy=u+z. Least Square Approximation​

Given a data set {(t1,y1),…,(tm,ym)}\{(t_1, y_1), \dots, (t_m, y_m)\}{(t1​,y1​),…,(tm​,ym​)}, where yiy_iyi​ is the observed value at time tit_iti​. If there is an essential linear relationship (y=ct+dy = ct + dy=ct+d), the goal is to find ccc and ddd such that the error E=∑i=1m(yi−cti−d)2E = \sum_{i=1}^m (y_i - ct_i - d)^2E=∑i=1m​(yi​−cti​−d)2 is minimized. This line is called the least squares line.

The problem is to find x0∈Fnx_0 \in \mathbb{F}^nx0​∈Fn such that EEE is minimized, i.e., ∥y−Ax0∥≤∥y−Ax∥\|y - Ax_0\| \le \|y - Ax\|∥y−Ax0​∥≤∥y−Ax∥ for all x∈Fnx \in \mathbb{F}^nx∈Fn.

Quick Summary​

This lecture covered crucial methods for working with Inner Product Spaces and linear data fitting. The Gram-Schmidt process provides a systematic way to convert any basis into an orthonormal one. We defined the orthogonal complement and explored the unique decomposition of vectors into components within a subspace and its complement, with the closest vector property. Finally, we introduced the least squares approximation method to find the best linear fit for data, derived its solution using the normal equations, and defined the adjoint operator in both matrix and abstract linear operator contexts.
