Systems of Linear Equations​

A system of mmm linear equations in nnn unknowns x1,…,xnx_1, \dots, x_nx1​,…,xn​ is expressed as:

{a11x1+⋯+a1nxn=b1a21x1+⋯+a2nxn=b2⋮am1x1+⋯+amnxn=bm\begin{cases} a_{11} x_1 + \dots + a_{1n} x_n = b_1 \\ a_{21} x_1 + \dots + a_{2n} x_n = b_2 \\ \quad \vdots \\ a_{m1} x_1 + \dots + a_{mn} x_n = b_m \end{cases}⎩⎨⎧​a11​x1​+⋯+a1n​xn​=b1​a21​x1​+⋯+a2n​xn​=b2​⋮am1​x1​+⋯+amn​xn​=bm​​

Here, aija_{ij}aij​ are the coefficients and bib_ibi​ are the constants, all real numbers. A linear equation implies its solution set has an affine space structure.

Matrix Notation​

A system of linear equations (E) can be concisely represented in matrix form as Ax=bA \mathbf{x} = \mathbf{b}Ax=b.

  • AAA is the coefficient matrix: An m×nm \times nm×n matrix where A=[aij]A = [a_{ij}]A=[aij​].
  • x\mathbf{x}x is the matrix of unknowns/variables: An n×1n \times 1n×1 column vector containing (x1,…,xn)T(x_1, \dots, x_n)^T(x1​,…,xn​)T.
  • b\mathbf{b}b is the matrix of constants: An m×1m \times 1m×1 column vector containing (b1,…,bm)T(b_1, \dots, b_m)^T(b1​,…,bm​)T. A system is homogeneous if b=0\mathbf{b} = \mathbf{0}b=0.

Elementary Row Operations​

These operations are fundamental for transforming matrices, particularly in Gaussian elimination, to find solutions or simplify systems.

  1. Interchanging two rows: Ri↔RjR_i \leftrightarrow R_jRi​↔Rj​.
  2. Multiplying a row by a non-zero real number: Ri→αRiR_i \rightarrow \alpha R_iRi​→αRi​, where α∈R\alpha \in \mathbb{R}α∈R and α≠0\alpha \neq 0α=0.
  3. Adding a multiple of one row to another row: Ri→Ri+βRjR_i \rightarrow R_i + \beta R_jRi​→Ri​+βRj​, where β∈R\beta \in \mathbb{R}β∈R. These operations reveal linear dependencies among rows, which can lead to zero rows in the row-reduced form of the matrix.

The Solution Space of a System​

A solution to a system (E) is an nnn-tuple α=(α1,…,αn)∈Rn\alpha = (\alpha_1, \dots, \alpha_n) \in \mathbb{R}^nα=(α1​,…,αn​)∈Rn that satisfies all equations in (E) when xi=αix_i = \alpha_ixi​=αi​.

  • SES_ESE​: Denotes the set of all solutions to the system (E).
  • SE0S_{E^0}SE0​: Denotes the set of all solutions to the corresponding homogeneous system E0E^0E0 (Ax=0A\mathbf{x} = \mathbf{0}Ax=0). Theorem: If αp\alpha_pαp​ is any particular solution to a non-homogeneous system (E), then the entire solution set SES_ESE​ is given by SE=SE0+αpS_E = S_{E^0} + \alpha_pSE​=SE0​+αp​. SE0S_{E^0}SE0​ possesses a vector space structure, while SES_ESE​ (for a non-homogeneous system) is an affine space.

Key Formulas​

  • System of Linear Equations (Matrix Form): Ax=bA \mathbf{x} = \mathbf{b}Ax=b
  • Elementary Row Operations:
    • Ri↔RjR_i \leftrightarrow R_jRi​↔Rj​
    • Ri→αRiR_i \rightarrow \alpha R_iRi​→αRi​, α≠0\alpha \neq 0α=0
    • Ri→Ri+βRjR_i \rightarrow R_i + \beta R_jRi​→Ri​+βRj​
  • Solution Set Relationship: SE=SE0+αpS_E = S_{E^0} + \alpha_pSE​=SE0​+αp​

Quick Summary​

Gaussian Elimination is a method for solving systems of linear equations, which can be represented compactly in matrix form Ax=bA \mathbf{x} = \mathbf{b}Ax=b. The process relies on elementary row operations to transform the matrix. The set of all solutions SES_ESE​ to a non-homogeneous system forms an affine space, directly related to the vector space of solutions SE0S_{E^0}SE0​ for its corresponding homogeneous system by adding a particular solution αp\alpha_pαp​.
