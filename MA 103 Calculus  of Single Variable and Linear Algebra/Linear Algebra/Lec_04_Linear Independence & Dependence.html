Vectors in Rn\mathbb{R}^nRn​

A vector in Rn\mathbb{R}^nRn is an nnn-tuple v=(v1,…,vn)v = (v_1, \dots, v_n)v=(v1​,…,vn​) where vi∈Rv_i \in \mathbb{R}vi​∈R. Rn\mathbb{R}^nRn is closed under vector addition and scalar multiplication, forming a vector space. The zero vector is 0=(0,0,…,0)\mathbf{0} = (0, 0, \dots, 0)0=(0,0,…,0).

Linear Combinations​

Definition: A linear combination of vectors v1,…,vmv_1, \dots, v_mv1​,…,vm​ is a vector of the form λ1v1+⋯+λmvm\lambda_1 v_1 + \dots + \lambda_m v_mλ1​v1​+⋯+λm​vm​, where λ1,…,λm\lambda_1, \dots, \lambda_mλ1​,…,λm​ are real numbers (scalars).

Linear Independence and Dependence​

Definition: Vectors v1,…,vmv_1, \dots, v_mv1​,…,vm​ are linearly independent (L.I.) if the equation λ1v1+⋯+λmvm=0\lambda_1 v_1 + \dots + \lambda_m v_m = \mathbf{0}λ1​v1​+⋯+λm​vm​=0 implies that λ1=⋯=λm=0\lambda_1 = \dots = \lambda_m = 0λ1​=⋯=λm​=0. Otherwise, the vectors are linearly dependent (L.D.). If vectors are L.D., at least one scalar λi\lambda_iλi​ is non-zero, meaning one vector can be expressed as a linear combination of the others.

Rank and Linear Independence​

Vectors v1,…,vmv_1, \dots, v_mv1​,…,vm​ are linearly independent if and only if the homogeneous system of linear equations resulting from λ1v1+⋯+λmvm=0\lambda_1 v_1 + \dots + \lambda_m v_m = \mathbf{0}λ1​v1​+⋯+λm​vm​=0 has only the trivial solution. In matrix form Aλ=0A\mathbf{\lambda} = \mathbf{0}Aλ=0 (where columns of AAA are viv_ivi​), this occurs if and only if rank(A)=m\text{rank}(A) = mrank(A)=m. There can be at most nnn linearly independent vectors in Rn\mathbb{R}^nRn.

Standard Basis Vectors​

In Rn\mathbb{R}^nRn, the standard basis vectors are e1=(1,0,…,0)e_1 = (1, 0, \dots, 0)e1​=(1,0,…,0), e2=(0,1,…,0)e_2 = (0, 1, \dots, 0)e2​=(0,1,…,0), …\dots…, en=(0,0,…,1)e_n = (0, 0, \dots, 1)en​=(0,0,…,1). These vectors are linearly independent, and any vector v=(a1,…,an)∈Rnv = (a_1, \dots, a_n) \in \mathbb{R}^nv=(a1​,…,an​)∈Rn can be uniquely written as a linear combination v=a1e1+⋯+anenv = a_1 e_1 + \dots + a_n e_nv=a1​e1​+⋯+an​en​.

Spanning Sets​

Definition: For a vector subspace WWW of Rn\mathbb{R}^nRn, a set of vectors {v1,…,vm}⊆W\{v_1, \dots, v_m\} \subseteq W{v1​,…,vm​}⊆W is a spanning set of WWW if every vector v∈Wv \in Wv∈W can be expressed as a linear combination of v1,…,vmv_1, \dots, v_mv1​,…,vm​. The standard basis vectors {e1,…,en}\{e_1, \dots, e_n\}{e1​,…,en​} form a spanning set for Rn\mathbb{R}^nRn.

Basis of a Vector Subspace​

Definition: A subset SSS of a vector subspace WWW is called a basis of WWW if SSS is both linearly independent and spans WWW. Theorem: Any two bases of a vector subspace of Rn\mathbb{R}^nRn have the same cardinality (the same number of elements).

Dimension of a Vector Subspace​

Definition: The dimension of a vector subspace is the cardinality (number of elements) of any of its bases. For example, R3\mathbb{R}^3R3 has dimension 3, and a subspace defined by x+y+z=0x+y+z=0x+y+z=0 in R3\mathbb{R}^3R3 has dimension 2.

Quick Summary​

This lecture clarified linear independence and dependence, linking them to the trivial solution of homogeneous linear systems and the rank of a matrix. Key concepts include linear combinations, the zero vector, and the standard basis of Rn\mathbb{R}^nRn. A basis was defined as a linearly independent spanning set for a vector subspace, and the unique cardinality of any basis for a given subspace was introduced as its dimension.
