Homogeneous Systems and Rank​

Definition: System of Linear Equations A system Ax=bAx=bAx=b involves a coefficient matrix AAA, a vector of unknowns xxx, and a constant vector bbb.

Definition: Homogeneous System A linear system is homogeneous if b=0b=0b=0, i.e., Ax=0Ax=0Ax=0. This system always has the zero solution (x=0x=0x=0).

Theorem: Non-Zero Solutions for Homogeneous Systems A homogeneous system Ax=0Ax=0Ax=0 has a non-zero solution if and only if:

  1. rank(A)\text{rank}(A) rank(A)<n nn, where nnn is the number of unknowns (columns of AAA).
  2. The column vectors of AAA are linearly dependent.

Definition: Rank of a Matrix The rank of a matrix AAA, denoted rank(A)\text{rank}(A)rank(A), is defined as: rank(A)=row-rank(A)=column-rank(A)\text{rank}(A) = \text{row-rank}(A) = \text{column-rank}(A)rank(A)=row-rank(A)=column-rank(A) The rank satisfies rank(A)≤min⁡{m,n}\text{rank}(A) \leq \min\{m, n\}rank(A)≤min{m,n}, where mmm is the number of rows and nnn is the number of columns.

Existence of Solutions for Ax=bAx=bAx=b​

Theorem: Consistency Condition (Rouché-Capelli / Rank Theorem) A system of linear equations Ax=bAx=bAx=b has at least one solution if and only if the rank of the coefficient matrix AAA is equal to the rank of the augmented matrix (A∣b)(A|b)(A∣b): rank(A)=rank(A∣b)\text{rank}(A) = \text{rank}(A|b)rank(A)=rank(A∣b) The augmented matrix (A∣b)(A|b)(A∣b) is formed by appending the column vector bbb to the matrix AAA.

Structure of Solution Sets​

The solution set for a consistent system Ax=bAx=bAx=b can be expressed as the sum of a particular solution xpx_pxp​ to Ax=bAx=bAx=b and the general solution S0S_0S0​ to the corresponding homogeneous system Ax=0Ax=0Ax=0: S={x∣Ax=b}={xp+xh∣Axp=b,Axh=0}S = \{x \mid Ax=b\} = \{x_p + x_h \mid Ax_p=b, Ax_h=0 \}S={x∣Ax=b}={xp​+xh​∣Axp​=b,Axh​=0} The solution set S0={x∣Ax=0}S_0 = \{x \mid Ax=0\}S0​={x∣Ax=0} is a vector subspace of Rn\mathbb{R}^nRn. A basis for S0S_0S0​ consists of linearly independent vectors that span the null space of AAA, and its dimension is the nullity of AAA, given by nullity(A)=n−rank(A)\text{nullity}(A) = n - \text{rank}(A)nullity(A)=n−rank(A).

Linear Transformations (Operators)​

Definition: Linear Transformation A function T:Rn→RmT: \mathbb{R}^n \to \mathbb{R}^mT:Rn→Rm is called a linear transformation (or linear operator) if, for all vectors x,y∈Rnx, y \in \mathbb{R}^nx,y∈Rn and all scalars λ∈R\lambda \in \mathbb{R}λ∈R, it satisfies the following two properties:

  1. Additivity: T(x+y)=T(x)+T(y)T(x+y) = T(x) + T(y)T(x+y)=T(x)+T(y)
  2. Homogeneity: T(λx)=λT(x)T(\lambda x) = \lambda T(x)T(λx)=λT(x)

Properties of Linear Transformations

  • T(0)=0T(0) = 0T(0)=0 (The zero vector in Rn\mathbb{R}^nRn maps to the zero vector in Rm\mathbb{R}^mRm).
  • T(−x)=−T(x)T(-x) = -T(x)T(−x)=−T(x).
  • Linear transformations map lines passing through the origin in Rn\mathbb{R}^nRn to lines passing through the origin in Rm\mathbb{R}^mRm.

Matrix Representation of a Linear Transformation Any linear transformation T:Rn→RmT: \mathbb{R}^n \to \mathbb{R}^mT:Rn→Rm can be represented by an m×nm \times nm×n matrix AAA. If e1,e2,…,ene_1, e_2, \ldots, e_ne1​,e2​,…,en​ are the standard basis vectors for Rn\mathbb{R}^nRn, then the matrix AAA has its columns given by T(e1),T(e2),…,T(en)T(e_1), T(e_2), \ldots, T(e_n)T(e1​),T(e2​),…,T(en​). Thus, T(x)=AxT(x) = AxT(x)=Ax.

Key Formulas​

  • Homogeneous System Non-Zero Solution Condition: Ax=0 has non-zero solution   ⟺  rank(A)Ax=0 \text{ has non-zero solution } \iff \text{rank}(A) Ax=0 has non-zero solution ⟺rank(A)<n nn.
  • Consistency Condition for Ax=bAx=bAx=b: Ax=b has solution   ⟺  rank(A)=rank(A∣b)Ax=b \text{ has solution } \iff \text{rank}(A) = \text{rank}(A|b)Ax=b has solution ⟺rank(A)=rank(A∣b).
  • Structure of Solution Set: S={xp+xh∣Axp=b,Axh=0}S = \{ x_p + x_h \mid Ax_p=b, Ax_h=0 \}S={xp​+xh​∣Axp​=b,Axh​=0}, where xpx_pxp​ is a particular solution and xhx_hxh​ is a solution to the homogeneous system.
  • Linear Transformation Properties:
    • T(x+y)=T(x)+T(y)T(x+y) = T(x) + T(y)T(x+y)=T(x)+T(y)
    • T(λx)=λT(x)T(\lambda x) = \lambda T(x)T(λx)=λT(x)
  • Matrix Representation: For a linear T:Rn→RmT: \mathbb{R}^n \to \mathbb{R}^mT:Rn→Rm, T(x)=AxT(x) = AxT(x)=Ax, where A=[T(e1) T(e2) … T(en)]A = [T(e_1) \ T(e_2) \ \dots \ T(e_n)]A=[T(e1​) T(e2​) … T(en​)].

Quick Summary​

Homogeneous systems Ax=0Ax=0Ax=0 always have the zero solution; non-zero solutions exist if rank(A)\text{rank}(A)rank(A) is less than the number of unknowns (nnn), implying linearly dependent column vectors. For non-homogeneous systems Ax=bAx=bAx=b, a solution exists if and only if rank(A)=rank(A∣b)\text{rank}(A) = \text{rank}(A|b)rank(A)=rank(A∣b). The general solution to Ax=bAx=bAx=b is the sum of a particular solution and the general solution to Ax=0Ax=0Ax=0. Linear transformations T:Rn→RmT: \mathbb{R}^n \to \mathbb{R}^mT:Rn→Rm preserve vector addition and scalar multiplication, satisfy T(0)=0T(0)=0T(0)=0, and can always be represented by an m×nm \times nm×n matrix whose columns are the images of the standard basis vectors.
