Multi-Layer Perceptrons and the Paradigm Shift

The Paradigm Shift in Machine Learning​

Historically, machine learning involved a two-step process where experts designed "hand-crafted" features (e.g., counting horizontal lines or pixel intensity for digit recognition) which were then fed into a trainable classifier.

  1. Traditional Machine Learning: Input →\rightarrow→ Hand-Crafted Feature Extractor →\rightarrow→ Trainable Classifier →y^\rightarrow \hat{y}→y^​.
  2. Neural Network Paradigm: Input →\rightarrow→ Trainable Feature Learner (Hierarchical) →\rightarrow→ Trainable Classifier →y^\rightarrow \hat{y}→y^​.

In NNs, the model automatically learns Low-Level Features (edges, blobs) in early layers and composes them into High-Level Features (shapes, objects) in deeper layers. This entire pipeline is trainable end-to-end.

The Artificial Perceptron​

Developed by Rosenblatt in the 1960s (inspired by McCulloch & Pitts), the perceptron is the simplest form of an artificial neuron.

Mathematical Definition​

A perceptron takes multiple binary (or real) inputs x1,x2,…,xdx_1, x_2, \dots, x_dx1​,x2​,…,xd​, applies weights wiw_iwi​, adds a bias bbb, and passes the sum through a step function.

y^={0if ∑wixi+b≤01if ∑wixi+b>0\hat{y} = \begin{cases} 0 & \text{if } \sum w_i x_i + b \leq 0 \\ 1 & \text{if } \sum w_i x_i + b > 0 \end{cases}y^​={01​if ∑wi​xi​+b≤0if ∑wi​xi​+b>0​

Components of a Neuron​

  1. Summation (Σ\SigmaΣ): The linear combination of inputs and weights plus bias: z=∑wixi+bz = \sum w_i x_i + bz=∑wi​xi​+b.
  2. Activation (∫\int∫ or ggg ): A non-linear function applied to zzz. For the original perceptron, this is the Heaviside Step Function (or sign function).

Linear Separability​

  • Linearly Separable: Data points can be perfectly divided by a single hyperplane (e.g., AND, OR).
  • Non-Linearly Separable: Data points cannot be separated by a single line (e.g., XOR).

The XOR Problem​

A single perceptron cannot solve XOR because the decision boundary required is non-linear.

  • Solution 1: Hand-craft non-linear features (e.g., x1xˉ2x_1 \bar{x}_2x1​xˉ2​ and xˉ1x2\bar{x}_1 x_2xˉ1​x2​).
  • Solution 2: Use a Multi-Layer Perceptron (MLP) to learn these feature combinations automatically.

Activation Functions​

To use Gradient Descent (Backpropagation), we need differentiable non-linearities, as the step function's derivative is zero almost everywhere.

Standard Activations​

  1. Sigmoid: g(z)=11+e−zg(z) = \frac{1}{1 + e^{-z}}g(z)=1+e−z1​
    • Output range: (0,1)(0, 1)(0,1).
    • Useful for probabilistic estimates.
  2. Tanh: g(z)=ez−e−zez+e−zg(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}}g(z)=ez+e−zez−e−z​
    • Output range: (−1,1)(-1, 1)(−1,1).

    • Game Changer: Default choice for modern deep networks.
    • Resolves vanishing gradient issues for z>0z > 0z>0.
  4. Leaky ReLU: g(z)=max⁡(αz,z)g(z) = \max(\alpha z, z)g(z)=max(αz,z) where α→0\alpha \to 0α→0.
    • Ensures the neuron "learns" even for z<0z < 0z<0.

Multi-Layer Perceptron (MLP) Architecture​

An MLP consists of an Input Layer (Layer 0), one or more Hidden Layers, and an Output Layer.

Notation​

  • LLL: Total number of layers.
  • N[l]N^{[l]}N[l]: Number of units (nodes) in layer lll.
  • a[l]a^{[l]}a[l]: Activation vector of layer lll (where a[0]=xa^{[0]} = xa[0]=x).
  • W[l]W^{[l]}W[l]: Weight matrix for layer lll.
  • b[l]b^{[l]}b[l]: Bias vector for layer lll.

Layerwise Computation (Vectorized)​

For a specific layer lll, the linear combination z[l]z^{[l]}z[l] and activation a[l]a^{[l]}a[l] are calculated as:

  • z[l]=W[l]a[l−1]+b[l]z^{[l]} = W^{[l]} a^{[l-1]} + b^{[l]}z[l]=W[l]a[l−1]+b[l]
  • a[l]=g(z[l])a^{[l]} = g(z^{[l]})a[l]=g(z[l])

Matrix Dimensions​

To ensure valid matrix multiplication:

  • a[l]∈RN[l]×1a^{[l]} \in \mathbb{R}^{N^{[l]} \times 1}a[l]∈RN[l]×1
  • W[l]∈RN[l]×N[l−1]W^{[l]} \in \mathbb{R}^{N^{[l]} \times N^{[l-1]}}W[l]∈RN[l]×N[l−1]
  • b[l]∈RN[l]×1b^{[l]} \in \mathbb{R}^{N^{[l]} \times 1}b[l]∈RN[l]×1

Quick Summary​

  • Paradigm Shift: Moved from hand-crafted feature engineering to hierarchical, automated feature learning.
  • The Neuron: Computes a weighted sum of inputs plus bias, followed by a non-linear activation.
  • Linear Separability: Single perceptrons can solve AND/OR/NAND but fail on XOR.
  • Activation Functions: ReLU is the modern default; Sigmoid/Tanh are used for specific probabilistic or centered data needs.
  • Vectorization: Forward propagation is expressed as a[l]=g(W[l]a[l−1]+b[l])a^{[l]} = g(W^{[l]}a^{[l-1]} + b^{[l]})a[l]=g(W[l]a[l−1]+b[l]), where W[l]W^{[l]}W[l] has shape (N[l]×N[l−1])(N^{[l]} \times N^{[l-1]})(N[l]×N[l−1]).
