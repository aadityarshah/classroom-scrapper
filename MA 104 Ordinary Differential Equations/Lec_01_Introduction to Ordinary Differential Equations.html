Basic Concepts: Modeling​

A physical system is translated into a mathematical model, which is then solved mathematically to obtain a physical interpretation.

Ordinary Differential Equations (ODEs)​

Definition: Ordinary Differential Equation (ODE)​

An ordinary differential equation (ODE) is an equation that contains one or more derivatives of an unknown function, usually denoted by y(x)y(x)y(x) (or y(t)y(t)y(t) when time ttt is the independent variable).

Definition: Order of an ODE​

An ODE is said to be of order nnn if the nthn^{th}nth derivative of the unknown function yyy is the highest derivative of yyy in the equation.

First-Order ODEs​

Equations of this type involve only the first derivative and may include yyy along with given functions of xxx.

Implicit Form of First-Order ODE​

F(x,y,y′)=0F(x, y, y') = 0F(x,y,y′)=0

Explicit Form of First-Order ODE​

y′=f(x,y)y' = f(x, y)y′=f(x,y)

Definition: Solution of a First-Order ODE​

A function y=h(x)y = h(x)y=h(x) is called a solution of F(x,y,y′)=0F(x, y, y') = 0F(x,y,y′)=0 on an interval aa a<xxx<b bb if:

  • hhh is differentiable on (a,b)(a, b)(a,b),
  • Substitution of y=h(x)y = h(x)y=h(x) and y′=h′(x)y' = h'(x)y′=h′(x) into the equation gives an identity.

Definition: Solution Curve / Integral Curve​

The graph of a solution h(x)h(x)h(x) is called a solution curve or an integral curve.

Solutions to ODEs and Initial Value Problems (IVPs)​

Definition: Particular Solution​

A particular solution is obtained from the general solution by using an initial condition y(x0)=y0y(x_0) = y_0y(x0​)=y0​, which determines the constant CCC. Geometrically, the solution curve should pass through (x0,y0)(x_0, y_0)(x0​,y0​).

Definition: Initial Value Problem (IVP)​

An ODE y′=f(x,y)y' = f(x, y)y′=f(x,y) together with an initial condition y(x0)=y0y(x_0) = y_0y(x0​)=y0​ is called an initial value problem (IVP).

Existence and Uniqueness of Solutions for IVPs​

A first-order IVP may exhibit:

  1. No solution.
  2. A unique solution.
  3. Infinitely many solutions. This motivates the study of conditions under which solutions to an IVP exist and are unique.

Homogeneous First-Order ODEs​

Definition: Homogeneous Function of Degree nnn​

A function f(x,y)f(x, y)f(x,y) is called homogeneous of degree nnn if f(tx,ty)=tnf(x,y)f(tx, ty) = t^n f(x, y)f(tx,ty)=tnf(x,y).

Homogeneous ODE Structure​

If y′=f(x,y)y' = f(x, y)y′=f(x,y) can be written as y′=g(y/x)y' = g(y/x)y′=g(y/x), then it is a homogeneous ODE. Such ODEs can be solved using the substitution u=y/xu = y/xu=y/x. This implies y=uxy = uxy=ux and y′=u′x+uy' = u'x + uy′=u′x+u.

Exact First-Order ODEs​

A differential equation of the form M(x,y) dx+N(x,y) dy=0M(x, y) \,dx + N(x, y) \,dy = 0M(x,y)dx+N(x,y)dy=0 is called exact if there exists a function u(x,y)u(x, y)u(x,y) such that du=M(x,y) dx+N(x,y) dydu = M(x, y) \,dx + N(x, y) \,dydu=M(x,y)dx+N(x,y)dy. The condition for exactness is ∂M∂y=∂N∂x\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}∂y∂M​=∂x∂N​.

If the equation is exact, its solution is implicitly given by u(x,y(x))=cu(x, y(x)) = cu(x,y(x))=c, for an arbitrary constant ccc.

Quick Summary​

Lecture 1 of MA 104 introduced Ordinary Differential Equations, defining them by the presence of derivatives of an unknown function and classifying them by order. We covered the implicit and explicit forms of first-order ODEs and the definition of a solution and solution curve. The lecture also touched upon how ODEs are used in modeling (e.g., exponential growth/decay), the concept of an Initial Value Problem (IVP), and noted that IVPs can have no, unique, or infinitely many solutions. Finally, the lecture briefly introduced homogeneous first-order ODEs and exact first-order ODEs, including their defining conditions and solution approaches.
