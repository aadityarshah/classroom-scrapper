First Order Linear ODEs​

Definition (Linear First Order ODE) A first order Ordinary Differential Equation (ODE) of the form y′=f(x,y)y' = f(x,y)y′=f(x,y) is called linear if f(x,y)f(x,y)f(x,y) depends linearly on the variable yyy, i.e., f(x,y)=a(x)y+b(x)f(x,y) = a(x)y + b(x)f(x,y)=a(x)y+b(x).

Definition (Standard Form) The standard form of a linear first order ODE is: y′+p(x)y=r(x)y' + p(x)y = r(x)y′+p(x)y=r(x)

Simpler Case (Constant Coefficients) If p(x)=pp(x) = pp(x)=p and r(x)=rr(x) = rr(x)=r (constants), the equation is y′+py=ry' + py = ry′+py=r. The solution is given by: y=Ce−px+rpy = C e^{-px} + \frac{r}{p}y=Ce−px+pr​ (where CCC is an arbitrary constant).

Solving First Order Linear ODEs (Integrating Factor Method)​

For the general standard form y′+p(x)y=r(x)y' + p(x)y = r(x)y′+p(x)y=r(x), the method of Leibniz (integrating factor) is used.

Definition (Integrating Factor) An integrating factor, μ(x)\mu(x)μ(x), for the linear first-order ODE y′+p(x)y=r(x)y' + p(x)y = r(x)y′+p(x)y=r(x) is a function such that when the ODE is multiplied by μ(x)\mu(x)μ(x), the left-hand side becomes the derivative of a product, ddx(μ(x)y)\frac{d}{dx}(\mu(x)y)dxd​(μ(x)y).

Theorem (Finding the Integrating Factor) The integrating factor μ(x)\mu(x)μ(x) for the ODE y′+p(x)y=r(x)y' + p(x)y = r(x)y′+p(x)y=r(x) is given by: μ(x)=e∫p(x)dx\mu(x) = e^{\int p(x)dx}μ(x)=e∫p(x)dx (A constant of integration can be chosen as C=1C=1C=1).

Theorem (General Solution of Linear First Order ODE) Given the ODE y′+p(x)y=r(x)y' + p(x)y = r(x)y′+p(x)y=r(x):

  1. Calculate the integrating factor μ(x)=e∫p(x)dx\mu(x) = e^{\int p(x)dx}μ(x)=e∫p(x)dx.
  2. Multiply the ODE by μ(x)\mu(x)μ(x): μ(x)y′+μ(x)p(x)y=μ(x)r(x)\mu(x)y' + \mu(x)p(x)y = \mu(x)r(x)μ(x)y′+μ(x)p(x)y=μ(x)r(x).
  3. The left-hand side is ddx(μ(x)y)\frac{d}{dx}(\mu(x)y)dxd​(μ(x)y). So, ddx(μ(x)y)=μ(x)r(x)\frac{d}{dx}(\mu(x)y) = \mu(x)r(x)dxd​(μ(x)y)=μ(x)r(x).
  4. Integrate both sides with respect to xxx: μ(x)y=∫μ(x)r(x)dx+C\mu(x)y = \int \mu(x)r(x)dx + Cμ(x)y=∫μ(x)r(x)dx+C.
  5. Solve for yyy: y=1μ(x)[∫μ(x)r(x)dx+C]y = \frac{1}{\mu(x)} \left[ \int \mu(x)r(x)dx + C \right]y=μ(x)1​[∫μ(x)r(x)dx+C]

Bernoulli Equation​

Definition (Bernoulli Equation) An equation of the form y′+Q(x)y=R(x)yαy' + Q(x)y = R(x)y^\alphay′+Q(x)y=R(x)yα, for a given fixed real number α\alphaα, is called a Bernoulli Differential Equation.

Note If α=0\alpha=0α=0 or α=1\alpha=1α=1, the Bernoulli equation is linear.

Theorem (Transformation to Linear ODE) Let α≠0,1\alpha \neq 0, 1α=0,1. The transformation u=y1−αu = y^{1-\alpha}u=y1−α reduces the Bernoulli equation y′+Q(x)y=R(x)yαy' + Q(x)y = R(x)y^\alphay′+Q(x)y=R(x)yα to a linear first-order ODE in uuu.

Method of Transformation

  1. Divide the Bernoulli equation by yαy^\alphayα: y′y−α+Q(x)y1−α=R(x)y'y^{-\alpha} + Q(x)y^{1-\alpha} = R(x)y′y−α+Q(x)y1−α=R(x).
  2. Let u=y1−αu = y^{1-\alpha}u=y1−α.
  3. Differentiate uuu with respect to xxx: u′=(1−α)y−αy′u' = (1-\alpha)y^{-\alpha}y'u′=(1−α)y−αy′.
  4. Substitute uuu and u′u'u′ into the modified equation:
    • From u′=(1−α)y−αy′u' = (1-\alpha)y^{-\alpha}y'u′=(1−α)y−αy′, we have y−αy′=11−αu′y^{-\alpha}y' = \frac{1}{1-\alpha}u'y−αy′=1−α1​u′.
    • Substitute this into step 1: 11−αu′+Q(x)u=R(x)\frac{1}{1-\alpha}u' + Q(x)u = R(x)1−α1​u′+Q(x)u=R(x).
    • This rearranges to the linear ODE: u′+(1−α)Q(x)u=(1−α)R(x)u' + (1-\alpha)Q(x)u = (1-\alpha)R(x)u′+(1−α)Q(x)u=(1−α)R(x)
  5. Solve this linear ODE for u(x)u(x)u(x) using the integrating factor method.
  6. Substitute back u=y1−αu = y^{1-\alpha}u=y1−α to find y(x)y(x)y(x).

Orthogonal Trajectories​

Definition (Orthogonal Trajectories) A new family of curves that intersect a given family of curves at right angles are called orthogonal trajectories of the given curves.

Definition (Angle of Intersection) The angle of intersection between two curves is defined as the angle between their tangent lines at the intersection point. For orthogonal curves, this angle is 90∘90^\circ90∘. If the slope of one curve is m1m_1m1​, and the slope of the orthogonal curve is m2m_2m2​, then m1m2=−1m_1 m_2 = -1m1​m2​=−1.

Method to Find Orthogonal Trajectories Given a one-parameter family of curves G(x,y,c)=0G(x,y,c)=0G(x,y,c)=0:

  1. Form the Differential Equation of the Given Family:

    • Eliminate the parameter ccc from G(x,y,c)=0G(x,y,c)=0G(x,y,c)=0 and its derivative to obtain the differential equation of the family: y′=f(x,y)y' = f(x,y)y′=f(x,y).
  2. Form the Differential Equation of the Orthogonal Trajectories:
    • The slope of an orthogonal trajectory is the negative reciprocal of the slope of the original family.
    • Replace y′y'y′ in y′=f(x,y)y' = f(x,y)y′=f(x,y) with −1y′-\frac{1}{y'}−y′1​ (or dydx\frac{dy}{dx}dxdy​ with −dxdy-\frac{dx}{dy}−dydx​).
